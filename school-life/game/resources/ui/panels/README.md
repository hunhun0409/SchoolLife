# UI Panel Textures

Pastel blue 9-slice panel textures for popup windows, dialogue boxes, and cards.

Recommended Godot settings:

- Use with `StyleBoxTexture`, `NinePatchRect`, or any texture-based panel control.
- Slice margins: `left=24`, `top=24`, `right=24`, `bottom=24`.
- Keep `expand_margin` at `0` unless a panel needs extra glow/shadow space.

Files:

- `panel_popup_blue.png`: default popup/window panel.
- `panel_dialogue_soft.png`: softer dialogue panel.
- `panel_card_light.png`: light card/content panel.
- `panel_accent_cyan.png`: stronger accented panel for important UI.
